Title: Math Painting
Description: An App that lets the user provide the start coordinates of geometrical shapes such as squares
and rectangles, their dimensions, their colors, and the program produces an image file with geometrical
shapes drawn in it.
Objects:
    Square:
            x
            y
            side
            color
            draw(canvas)

    Rectangle:
                x
                y
                width
                height
                color
                draw(canvas)

    Canvas:
            width
            height
            color
            make(imagepath)
